sesac 프로젝트 3차

### Q. 어떤 문제를 해결하는 프로젝트인가? [데이터 분석 AI] [지식 기반 QA 시스템]
A. 파편화된 시장 데이터와 뉴스 정보를 [데이터 분석 AI]와 [RAG 기반 QA 시스템]으로 통합하여, 초보 투자자의 데이터 기반 의사결정을 돕는 'AI 투자 비서'를 지향.

### Q. 기존 방식은 무엇인가?
A. 사용자가 개별 정보를 일일이 검색하고 해석하던 30분의 분석 과정을 1분 이내의 통합 리포트로 압축하여, 투자 판단의 효율성을 극대화.

### Q. 왜 LLM / Agent가 필요한가?
A. 비정형 데이터의 '맥락(Context)' 해석 능력으로 뉴스의 긍/부정 을 파악.

### Q. 주요 사용자
A. 정보 과잉에 지친 스마트 개미: 쏟아지는 뉴스 속에서 어떤 정보가 진짜 호재인지 가려낼 시간이 부족한 직장인 투자자.
   데이터 기반의 확신이 필요한 초보자: 단순한 감(Feeling)이 아니라, 차트 데이터와 뉴스 분석이 결합된 객관적인 근거를 보고 싶은 투자 입문자.

### Q. 핵심 기능
1. 하이브리드 데이터 분석: 한국투자증권 API의 실시간 가격 데이터와 뉴스 API의 텍스트 데이터를 결합하여 시장 상황을 입체적으로 분석합니다.
2. LLM 뉴스 감성 스코어링: 단순 키워드 매칭이 아닌, AI가 기사 맥락을 읽어 긍정/부정/중립 점수를 산출하여 투자 심리를 수치화합니다.
3. RAG 기반 전문가 인사이트: 증권사 리포트 등 신뢰할 수 있는 PDF 지식을 기반으로, 현재 시장 상황에 가장 적합한 투자 원칙을 인용해 답변합니다.
4. 세 줄 투자 요약 리포트: 기술 지표와 뉴스 분석 결과를 종합하여 '매수/매도/관망' 신호와 그 이유를 사용자가 이해하기 쉬운 자연어로 제공합니다.

### Q. 전체 서비스 흐름
1. 검색: 사용자가 궁금한 종목명 또는 종목 코드를 입력합니다.
1. 수집: Backend에서 해당 종목의 최근 주가와 관련 뉴스를 실시간으로 가져옵니다.
1. 분석: * 기술 분석: RSI, 이동평균선 등 기술적 지표를 계산합니다.
1. 감성 분석: LLM이 수집된 뉴스의 긍/부정 수치를 판단합니다.
1. RAG 연동: Vector DB에 저장된 전문가 리포트나 투자 가이드라인에서 현재 지표와 관련된 핵심 내용을 검색(Retrieval)합니다.
1. 생성: 모든 데이터(가격+뉴스+RAG 지식)를 LLM에 전달하여 최종 투자 의견과 근거를 담은 리포트를 생성합니다.
1. 출력: 웹 인터페이스를 통해 LLM의 답변을 사용자에게 보여줍니다.
